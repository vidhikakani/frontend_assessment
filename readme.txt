Prerequisites:
1. Download and install Node.js (https://nodejs.org/en/)
2. Download java 11+ (https://www.oracle.com/java/technologies/javase/jdk11-archive-downloads.html)
3. Follow steps from https://reactnative.dev/docs/environment-setup
4. Download and install Android Studio (https://developer.android.com/studio/?gclid=Cj0KCQiA0p2QBhDvARIsAACSOONWphUXRG0lucKsj606d1_2zgAQ2RGfENWwhCPMzFKchqU-Rslep-saAglBEALw_wcB&gclsrc=aw.ds)
5. Download and install XCode from apple store

Steps to run:
1. git clone https://github.com/vidhikakani/pop_frontend_assessment.git
2. cd pop_frontend_assessment
3. npm install 
4. npm run ios [to run on ios]
5. npm run android [to run on android]
6. npm run web [to run on web]
